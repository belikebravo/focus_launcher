## ğŸ› ï¸ Setup & Installation

### Prerequisites
-   Android Studio Koala or newer.
-   An Android device running Android 12+.

### Getting Started
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/belikebravo/focus_launcher.git
    ```
2.  **Open in Android Studio**:
    -   Select `Open` and navigate to the `focus_launcher` folder.
    -   Wait for Gradle sync to complete.
3.  **Build & Run**:
    -   Connect your device or start an emulator.
    -   Click **Run (Shift+F10)**.
    -   *Note: You may need to set Focus Launcher as your default home app in settings.*

### ğŸ”‘ AI Configuration (Gemini)
To enable dynamic Knowledge Nuggets, you need a Gemini API Key:
1.  Get a free API Key from [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  Open the app on your phone.
3.  Go to **Settings** (Swipe or Long press).
4.  Enter your **Gemini API Key**.
5.  Select topics you want to learn about!
